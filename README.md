# TubeShelf

<div id="top"></div>

YouTubeのチャンネルを探すときにいっぱいありすぎて困ったことはありませんか？私はあります。

今日はゲーム実況を見ようかな？と思ってYouTubeのトップページにいってもいつも聴いている音楽のチャンネルしかない！なんてことありますよね。

だから私は TubeShelf を作成しました。これで私の好きなゲーム実況者のチャンネルの一覧が簡単に見ることができます。また、ずっと昔に登録したShortsのチャンネル、ゲーマーだっけ？ミュージシャンだっけ？エンジニアだっけ？となってもTubeShelfでラベリングしたら簡単にわかります。

ぜひ体験してみてください。

## 使用方法

1. Chrome Web Storeから TubeShelf をダウンロードしてインストールする
2. YouTube を開く
3. TubeShelfをクリックして有効にする
