# デプロイ (Deployment)

このディレクトリでは、アプリケーションのリリース、公開手順、およびストア管理に関する情報を管理します。
本番環境（Chrome Web Store及びユーザー環境）への安全なデリバリーを確実にします。

## ドキュメント構成

本ディレクトリには、以下のドキュメントを配置します。

| ドキュメント       | ファイル名                                       | 内容概要                                                                                               |
| :----------------- | :----------------------------------------------- | :----------------------------------------------------------------------------------------------------- |
| **リリース手順書** | [`release_procedure.md`](./release_procedure.md) | バージョン採番ルール、ビルド、パッケージング、ストアへのアップロード手順を記述します。                 |
| **ストア掲載情報** | [`store_assets.md`](./store_assets.md)           | Chrome Web Storeに掲載する説明文、スクリーンショット、プロモーション画像の要件と管理場所を記述します。 |
| **リリースノート** | [`release_notes.md`](./release_notes.md)         | 各バージョンの変更点、新機能、バグ修正内容をユーザー向けに記録します。                                 |
