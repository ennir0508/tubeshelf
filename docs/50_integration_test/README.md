# 結合テスト (Integration Test)

このディレクトリでは、複数コンポーネントの連携や、システム全体の通しテストに関する情報を管理します。
機能要件が正しく満たされているかを検証します。

## ドキュメント構成

本ディレクトリには、以下のドキュメントを配置します。

| ドキュメント         | ファイル名                                     | 内容概要                                                                               |
| :------------------- | :--------------------------------------------- | :------------------------------------------------------------------------------------- |
| **結合テスト計画書** | [`integration_plan.md`](./integration_plan.md) | 結合テストの実施スケジュール、環境、対象範囲、前提条件を定義します。                   |
| **テストシナリオ**   | [`test_scenarios.md`](./test_scenarios.md)     | ユーザー操作を想定した一連の処理フロー（ユースケース）と、期待される結果を記述します。 |
| **テスト実施報告書** | [`test_report.md`](./test_report.md)           | テストの実施結果、発見されたバグ、修正状況のサマリを記録します。                       |
