# 画面設計書 (UI/UX Requirements Definition)

## 1. 全体レイアウト

### 1.1. サイドバー (Sidebar)
- **配置**: YouTube画面左側のナビゲーションメニュー（通常「ホーム」「ショート」「登録チャンネル」などが並ぶエリア）内に統合、またはそのエリアを置換して表示。
- **構成**:
    - **ヘッダー**: 「TubeShelf」タイトル、設定ボタン（歯車アイコン）。
    - **カテゴリツリー**: フォルダアイコン、カテゴリ名、開閉矢印、未読バッジ。
    - **未分類リスト**: カテゴリに属さないチャンネルの一覧。

## 2. インタラクション (操作)

### 2.1. ドラッグ＆ドロップ
- **チャンネル移動**: チャンネル名をドラッグし、カテゴリ名の上でドロップすると移動。
- **並べ替え**: カテゴリ同士の順序変更もD&Dで可能とする。
- **視覚フィードバック**: ドロップ可能な領域（カテゴリ）にホバー時、背景色を変更して強調表示。

### 2.2. コンテキストメニュー (右クリックメニュー)
- **カテゴリ上**: 「名前の変更」「削除」「子カテゴリ作成」「上に移動」「下に移動」。
- **チャンネル上**: 「カテゴリから除外」「（カテゴリ選択）へ移動」。

## 3. ポップアップ画面 (Extension Popup)

### 3.1. 機能
- ブラウザ右上の拡張機能アイコンクリック時に表示。
- バージョン情報表示。
- 「設定画面を開く」リンク。
- 簡易的なステータス表示（「同期完了」「N個のカテゴリ」など）。

## 4. 設定画面 (Options Page)

### 4.1. データ管理
- カテゴリデータのエクスポート（JSON形式）。
- インポート機能。
- 初期化（全データ削除）ボタン。

### 4.2. 表示設定
- サイドバーの表示幅調整（もし可能なら）。
- テーマ設定（YouTubeのダークモード設定に自動追従）。
